Welcome to Social Circle

A lightweight, responsive demo that showcases posts, user profiles, and comments fetched from public APIs (DummyJSON). Use the navigation above to explore.



How it Works ?

  • Posts: Loads posts dynamically with pagination. Click a username to see a profile modal.
  • Comments: Fetches comments per post. Shows “No comments available !” when empty.
  • Contact: Validates Form: Name (no numbers), Email (must have “@” and “.”), and a confirmation checkbox that enables the Send button.


Tip: Use the Posts tab to start browsing!
